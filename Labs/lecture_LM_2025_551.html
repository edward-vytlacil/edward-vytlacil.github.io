Linear Combination of Coefficients

Recall Asymptotic Distribution of OLS

Under regularity conditions,
\[\sqrt{N} \left( \widehat \beta_N - \beta \right) \stackrel{d}{\rightarrow} Z,~~~Z \sim N(0,\Sigma),\] which by CMT implies that, for any continuous function \(g\), \[g(\sqrt{N} \left( \widehat \beta_N - \beta \right) \stackrel{d}{\rightarrow} g(Z),~~~Z \sim N(0,\Sigma),\] and thus, for \(a\) any \((K+1)\times 1\) vector of constants, \[\sqrt{N} \left( a^{\prime} \widehat \beta_N - a^{\prime} \beta \right) \stackrel{d}{\rightarrow} N(0, a^{\prime} \Sigma a).\]

Standard Errors for Linear Combination of \(\beta\)

\[\sqrt{N} \left( a^{\prime} \widehat \beta_N - a^{\prime} \beta \right) \stackrel{d}{\rightarrow} N(0, a^{\prime} \Sigma a).\]

  • Let \(\widehat \Sigma_N\) denote a consistent estimator of \(\Sigma\), \[\widehat \Sigma_N \stackrel{p}{\rightarrow} \Sigma ~~\Rightarrow ~~ a^{\prime} \widehat \Sigma_N a \stackrel{p}{\rightarrow} a^{\prime} \Sigma_N a.\]

    \[ \mbox{s.e.}(a^{\prime} \widehat \beta_N) = \widehat \omega_N / \sqrt{N} ~~\mbox{where}~~ \widehat \omega^2_N = a^{\prime} \widehat \Sigma_N a \]

Test Null for Linear Combination of \(\beta\).

Consider \[H_0: \alpha^{\prime} \beta = \alpha^{\prime} b,~~~\mbox{vs}~~ H_1: \alpha^{\prime} \beta \ne \alpha^{\prime} b.\]

  • Define test statistic: \[T_N = \biggl | \frac{a^{\prime} \widehat \beta_N - a^{\prime} b}{\mbox{s.e.}(a^{\prime} \widehat \beta_N)} \biggr |.\]

Test Null for Linear Combination of \(\beta\).

\(T_N \stackrel{d}{\rightarrow} N(0,1)\) under \(H_0\).

For test at (asymptotic) level \(\alpha\),

  • reject \(H_0\) if \(T_N \ge c_{1-\alpha/2}\) where \(c_{1-\alpha/2} = \Phi^{-1}(1-\alpha/2)\).

  • equivalently, define \(p = 2 (1- \Phi(T_N))\), reject null if \(p \le \alpha\).

CI for Linear Combination of \(\beta\)

Define \[T_N(\alpha^{\prime} b) = \biggl | \frac{a^{\prime} \widehat \beta_N - a^{\prime} b}{\mbox{s.e.}(a^{\prime} \widehat \beta_N)} \biggr |.\]

Invert (asymptotic) level \(\alpha\) test to form \(1-\alpha\) asymptotic CI: \[ \widehat C_N = \{\alpha^{\prime} b: T(\alpha^{\prime} b) \leq c_{1-\alpha/2}\} \\ = \left[ a^{\prime} \widehat\beta_N - c_{1-\alpha/2} \times \mbox{s.e.}(a^{\prime} \widehat \beta_N), a^{\prime} \widehat\beta_N + c_{1-\alpha/2} \times \mbox{s.e.}(a^{\prime} \widehat \beta_N) \right].\]

CI for Linear Combination of \(\beta\)

\[ \widehat C_N = \{a^{\prime} b: T(a^{\prime} b) \leq c_{1-\alpha/2}\}\]

\[\Rightarrow \Pr(a^{\prime} \beta \in \widehat{C}_N) = \Pr(T_N(a^{\prime} \beta) \leq c_{1-\alpha/2}) \rightarrow 1-\alpha,\]

  • Thus \(\widehat C_N\) is asymptotic \(1-\alpha\) CI on \(\alpha^{\prime} \beta\).

CI for Linear Combination of \(\beta\)

\[ \widehat C_N = \{a^{\prime} b: T(a^{\prime} b) \leq c_{1-\alpha/2}\}\]

\[\Rightarrow \Pr(a^{\prime} \beta \in \widehat{C}_N) = \Pr(T_N(a^{\prime} \beta) \leq c_{1-\alpha/2}) \rightarrow 1-\alpha,\]

  • Can use \(\widehat C_N\) to test null \(H_0: \alpha^{\prime} \beta = \alpha^{\prime} b\) for any \(\alpha^{\prime} b\).

    • Reject \(H_0: \alpha^{\prime} \beta = \alpha^{\prime} b\) for any \(\alpha^{\prime} b \not \in \widehat{C}_N\).

    • Fail to reject \(H_0: \alpha^{\prime} \beta = \alpha^{\prime} b\) for any \(\alpha^{\prime} b \in \widehat{C}_N\).

Digression: Failure to Reject \(H_0\) as Evidence \(H_0\) is True

A common misconception is to interpret acceptance of the null hypothesis as evidence that the null is true.

  • Such an interpretation is only warranted if the power of the test against alternatives of interest is high.
  • If the test has low power against an alternative of interest, then it is unlikely to reject the false null when that alternative it true, and we should expect the test to incorrectly accept the null even when that alternative is true instead.

  • Figure from Amrhein et al. (2019) who provide a discussion of this misconception across fields.

  • Ziliak and McCloskey made related points in a series of publications, arguing against asterisk econometrics. See, e.g., Size matters: the standard error of regressions in the American Economic Review

Digression: Failure to Reject \(H_0\) as Evidence \(H_0\) is True

  • CI shows range of null hypotheses that are not rejected.

  • If estimating effect, how to interpret evidence if:

    • CI includes 0 but also values far from 0?

    • CI includes 0 and only values close to 0?

      • Sometimes called precisely estimated 0 in applied work.

Example 1: Progresa

\[ Y_{i} = \beta_0 + \beta_1 \mbox{Treat}_i + \beta_2 \mbox{Girl}_i + \beta_3 \mbox{Treat}_i \times \mbox{Girl}_i + \epsilon_i.\] Consider testing null of zero average effect for both boys and girls: \[~H_0: \beta_1 =0~ ~ \mbox{and} ~~ \beta_3=0,~~~\mbox{vs.}~~~~H_1: \beta_1 \ne 0~ ~ \mbox{or} ~~ \beta_3\ne 0.\]

  • Joint hypothesis test.
  • How different from multiple hypothesis test?

Consider \[H_0: R\beta=c ~~\mbox{vs}~~H_1: R\beta \neq c,\] where \(R\) is a \(p \times (K+1)\)-dimensional constant matrix and \(c\) is a \(p\times 1\) vector.

Suppose \(\widehat \Sigma_N \stackrel{p}{\rightarrow} \Sigma\). Then

\[\begin{align*}\sqrt{N}(\widehat\beta_N &- \beta) \stackrel{d}{\rightarrow} N(0, \Sigma )\\ & \Rightarrow~~ \sqrt{N} (R \widehat\beta_N- R \beta) \to_d N(0, R\Sigma R')\\ & \Rightarrow~~ N (R \widehat\beta_N- R \beta)^{\prime}(R \widehat \Sigma_N R^{\prime})^{-1} (R \widehat\beta_N- R \beta) \stackrel{d}{\rightarrow} \chi^2_p\end{align*}\]

Let \[ \begin{align*} T_N &= N (R \widehat\beta_N- c)^{\prime}(R \widehat \Sigma_N R^{\prime})^{-1} (R \widehat\beta_N-c)\\ & = (R \widehat\beta_N- c)^{\prime}(R (\widehat \Sigma_N/N) R^{\prime})^{-1} (R \widehat\beta_N-c). \end{align*}\]

\[T_N \stackrel{d} {\rightarrow} \chi^2_p~~~ \mbox{under}~ H_0: R\beta=c\]

\[T_N \stackrel{d} {\rightarrow} \chi^2_p~~~ \mbox{under}~ H_0: R\beta=c\]

  • Reject null at asymptotic level \(\alpha\) if \(T_N > c_{p,1-\alpha}\), where \(c_{p,1-\alpha} = F_p^{-1}(1-\alpha), F_p\) cdf of \(\chi_p^2\).

  • \(p\)-value is \(1-F_p(T_N)\), invert for the CI.
